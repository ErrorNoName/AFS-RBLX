# SolarisUi Library

The SolarisUi library provides a comprehensive set of functions for creating user interfaces in Roblox. This library simplifies the process of creating and managing various UI components, allowing for easy customization and event handling.

## Available UI Elements

The following UI elements can be created using the SolarisUi library:

1. **Button**
   - **Properties**: 
     - Text
     - Size
     - Position
     - BackgroundColor
     - TextColor
     - Font
     - OnClick event
   - **Methods**: 
     - SetText
     - SetSize
     - SetPosition
     - SetBackgroundColor
     - SetTextColor
     - SetFont
     - ConnectOnClick

2. **TextLabel**
   - **Properties**: 
     - Text
     - Size
     - Position
     - TextColor
     - Font
     - BackgroundTransparency
   - **Methods**: 
     - SetText
     - SetSize
     - SetPosition
     - SetTextColor
     - SetFont

3. **TextBox**
   - **Properties**: 
     - PlaceholderText
     - Size
     - Position
     - TextColor
     - Font
     - BackgroundTransparency
   - **Methods**: 
     - SetPlaceholderText
     - SetSize
     - SetPosition
     - SetTextColor
     - SetFont
     - GetText

4. **Frame**
   - **Properties**: 
     - Size
     - Position
     - BackgroundColor
     - BorderSizePixel
   - **Methods**: 
     - SetSize
     - SetPosition
     - SetBackgroundColor
     - SetBorderSize

5. **ImageLabel**
   - **Properties**: 
     - Image
     - Size
     - Position
     - BackgroundTransparency
   - **Methods**: 
     - SetImage
     - SetSize
     - SetPosition

6. **UICorner**
   - **Properties**: 
     - CornerRadius
   - **Methods**: 
     - SetCornerRadius

7. **UIStroke**
   - **Properties**: 
     - Color
     - Thickness
     - Transparency
   - **Methods**: 
     - SetColor
     - SetThickness
     - SetTransparency

8. **UIPadding**
   - **Properties**: 
     - Padding
   - **Methods**: 
     - SetPadding

9. **UIListLayout**
   - **Properties**: 
     - FillDirection
     - HorizontalAlignment
     - VerticalAlignment
   - **Methods**: 
     - SetFillDirection
     - SetHorizontalAlignment
     - SetVerticalAlignment

10. **UIGridLayout**
    - **Properties**: 
      - CellSize
      - CellPadding
    - **Methods**: 
      - SetCellSize
      - SetCellPadding

## Usage

To use the SolarisUi library, simply require the `SolarisUi.lua` file in your Roblox project and start creating UI components using the provided functions. This library aims to streamline the UI creation process, making it easier for developers to build engaging interfaces for their games.
